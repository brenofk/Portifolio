Sobre mim

Olá, meu nome é Breno, tenho 23 anos e estou cursando Tecnologia em Análise e Desenvolvimento de Sistemas no Instituto Federal do Paraná (IFPR). Atualmente, estou no terceiro ano e me formando este ano.

Ao longo da minha jornada acadêmica, adquiri conhecimentos em diversas linguagens de programação, como Java, JavaScript, HTML, CSS e Python. Participei de projetos acadêmicos que ampliaram minhas habilidades técnicas, incluindo: Desenvolvimento de um banco de dados utilizando Python com Django e Django REST Framework. Criação de interfaces de login e cadastro de clientes em Java. Análise de dados utilizando a biblioteca Pandas no Python. Desenvolvimento do front-end de aplicações com HTML e CSS.
